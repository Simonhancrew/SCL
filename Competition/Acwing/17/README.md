复习一下组合数的公式
C[a][b] = C[a - 1][b] + C[a - 1][b - 1]
从a中选择b有两种情况：

第一种，已经拿出一个要选的了，C[a - 1][b - 1]

第二种，之前那个拿出来的不选，结果就是C[a - 1][b]

```
// C[a][b] = C[a - 1][b] + C[a - 1][b - 1]
void init(){
    for(int i = 0;i < N;i++){
        for(int j = 0;j <= i;j++){
            if(!j) C[i][j] = 1;
            else C[i][j] = (C[i - 1][j] + C[i - 1][j - 1]);
        }
    }
}
```
