## A*

BFS的优化，一种求最短路的算法，需要一个估价函数（启发函数），只需要遍历很少的路径就能够搜到结局

点数非常庞大的时候A*算法会有很大的优势

BFS中的队列会被换成优先队列，除了存储真实距离（起点到当前点）之外还要存储估价距离（从当前点到终点的猜测距离），排序的是整个估价 + 真实
