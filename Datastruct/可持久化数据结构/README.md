## Persistent data structure

可持久化数据结构的先决条件就是数据结构的拓扑结构不会发生改变

其可以保留每一个历史版本（大概率只记录每一个版本和前一个版本不同的节点），并且支持操作的不可变性。

可持久化可以分为两类：

1. 部分可持久化数据结构：
    > 所有的版本都可以访问，但是只有最新版本的可以修改

2. 完全可持久化数据结构：
    > 所有的版本都可以修改和访问，若支持将两个版本合并，则称之为，Confluently Persistent
